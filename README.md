### beautiful-web-ppt

技术栈：_Lerna + Vite + ES6 + Scss_

### PPT-Manager

========================================================

为 **_PPT-MAKE_** 提供**数据源**以及定制化 UI

功能点：

1. PPT 背景图制作（限制 100 张？）

- 提供风格以及色系选择，自动化生成
- 提供选色板，手动制作
- logo（icon + 文字，可选）
- 水印 （可选）
- 页码（待定，可选）

2. 首图

- 标题（必填项）
- 日期（选填）
- 汇报人（选填）
- 部门（选填）
- 自定义字段

3. 目录页面

- 大标题
- 小标题
- 描述

4. 尾图

- 随机生成结束语（从词库中取）
- 结束语（选填，默认 Thanks）

4. 内容图

- 大标题
- 描述

### PPT-MAKE

========================================================

从 PPT-Manage-System 获取数据源，自动生成 PPT 网页

reveal.js
